# En garde

AI-powered fencing posture coach built with SwiftUI, Vision, and AVFoundation.

## Structure
- MVVM architecture
- Global `AppState` injected with `@EnvironmentObject`
- Mocked camera/Vision pipeline ready for integration
- Exercise progression flow with unlock state
